# CandidateTracker

## Setup

1. Clone repository

    `git clone https://github.com/anthonytongUCB/CandidateTracker.git`

2. Select alias (might have to install some firebase stuff before using this command)

    `firebase use tracker`

3. Launch
   
    `firebase serve`


4. Access application
   
    `http://localhost:5000/`
